Project: VOLCAN-I-CITY: JS Code

VOLCAN-I-CITY is a 2D platform game with a Ku-Kia'i-Mauna-themed Sonic that runs and jumps to avoid lava splatter. He also collects wai(water) and kalo(taro) instead of coins.

Project: MaZe RuNNeR:JS Code

Maze Runner is a little maze I created in scratch. Basically the player moves the ball from point a to point b without touching the walls. That's it! There's nothing difficult. I used leopard js to convert the scratch game to javascript and leopard automatically uploaded the game to codesandbox. I'm still learning, so I hope to code the next project fully on my own.
